计入范围

测试期间，玩家购买【任意档位钻石】、【通行证】和【礼包】的金额将计入【累计充值总额】。

返利档位

游戏正式上线后，将根据测试期间同一账户的累计充值金额按比例返还钻石：

  • ≤ USD100：200% Diamond Rebate
  • > USD100：
    • Amount ≤ USD100 部分：200% Rebate
    • Excess Amount（超出部分）：100% Rebate
  • All decimal values rounded up（所有小数向上取整）

货币与钻石换算

每 USD0.99 对应 60 钻石。

官方示例

  • 玩家 a 累计 USD49.99 → 3000 × 2 = 6000 钻石
  • 玩家 b 累计 USD129.97 → 6000 × 2 + 1800 = 13800 钻石
本计算器采用的公式：钻石 = ceil(USD) × 60（即 USD 金额先向上取整到整数美元，再乘以 60）。
此公式是唯一能同时对齐官方两个示例数值的算法。
